# Async Programming Practice

A collection of exercises and practice projects to understand **asynchronous programming** concepts such as `Futures`, `async/await`, and API calls.  

This repository is focused on hands-on practice to strengthen asynchronous programming skills in **Dart/Flutter** (or other languages as needed).

---

##  Features

- Practice with `Futures` and `async/await`
- Handling asynchronous API calls
- Simulated real-world async tasks
- Small projects and exercises for learning
